1- Quem inventou o sistema de controle de versão Git e porquê ?
R: Linux Torvalds, que tambem é criador do Linux.   A principio sua criação era para ser de grade colaborativa, após suas relações com uma BitKeeper ser "desconstruida" devido a
BitKeeper começar a cobrar pelo seu uso, Linux viu a necessidade de desenvolver uma nova ferramenta baseada em lições aprendidas ao usar o Bitkeeper.

ao unir forças com o GitHub, nós fortalecemos nosso comprometimento com a liberdade do desenvolvedor."
